To use your custom spice shaker SVG icon:

1. Place your SVG file at: assets/icons/spice_shaker.svg
2. Make sure the SVG uses `fill="currentColor"` or no fill for color control.
3. Run `flutter pub get` after adding the asset.
4. The code will use SvgPicture.asset('assets/icons/spice_shaker.svg', color: ...)
